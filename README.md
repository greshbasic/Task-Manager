<h2> How to Run: </h2>
<ol>
  <li>After cloning, navigate to top of dir (<code>/Task-Manager</code>)</li>
  <li>Run <code>npm i && ng serve</code></li>
  <li>Head to <a href="http://localhost:4200/">localhost:4200</a></li>
</ol>
